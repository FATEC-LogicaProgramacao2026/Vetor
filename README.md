# Vetor

# 📦 Problema sem vetor

Imagine que precisamos guardar **4 notas de um aluno**.

# 📦 Usando Vetor

Com vetor podemos guardar tudo em **uma única variável**:

```
notas[4]
```

Agora cada nota fica em **uma posição do vetor**.

---

```
Indice:   1    2    3    4 
        
notas: │ 7  │ 8  │ 6  │ 9  │
        
```

Cada posição possui:

* **Índice (posição)**
* **Valor armazenado**
